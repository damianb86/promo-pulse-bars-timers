# Promo Pulse: Bars & Timers Product Spec - Stage 1

## Resumen

Promo Pulse: Bars & Timers es una app de campanas promocionales para Shopify.
Permite crear, publicar, segmentar, localizar y medir experiencias
promocionales en storefront, producto y carrito, usando extensiones oficiales
de Shopify y datos propios de la app.

## Objetivo de Stage 1

Construir una version funcional y publicable con un Campaign Builder central,
superficies promocionales clave, targeting basico, sincronizacion inicial con
descuentos nativos y analytics suficiente para evaluar rendimiento.

## Funcionalidades

- Campaign Builder central para crear y administrar campanas.
- Countdown Promo Bar.
- Product Page Timer.
- Cart Page Timer y Cart Drawer Timer.
- Free Shipping Goal Bar.
- Delivery Cutoff Timer.
- Discount Sync basico con descuentos nativos de Shopify.
- Low Stock Message basico.
- Product Badge simple.
- Templates por evento promocional.
- Live preview desktop/mobile.
- Onboarding guiado.
- Plan gating basico: Free, Starter, Growth y Pro.

## Targeting

- Producto.
- Coleccion.
- Pagina.
- Pais.
- Idioma.
- Dispositivo.
- Parametros UTM.
- Customer tag cuando Shopify lo permita en el contexto disponible.

## Multi-idioma storefront

Idiomas iniciales:

- Ingles.
- Espanol.
- Portugues BR.
- Frances.
- Aleman.

## Extensiones Shopify

- Theme App Extension para bloques de storefront.
- App Embed para cargas globales y superficies transversales.
- Web Pixel Extension para analytics basico.

## Analytics basico

Eventos iniciales esperados:

- Views.
- Clicks.
- Add to cart.
- Checkout started.
- Orders/revenue aproximado cuando sea posible atribuirlo con datos disponibles.

## Plan gating inicial

- Free: entrada limitada para probar una campana basica.
- Starter: superficies principales y targeting simple.
- Growth: targeting avanzado, templates y analytics ampliado.
- Pro: mayor capacidad, multi-idioma completo y controles avanzados.

Los limites de plan estan definidos para MVP; Shopify Billing real sigue como
integracion pendiente antes de publicacion.

## Fuera de alcance por ahora

- Automatizaciones avanzadas de diseno fuera del editor visual actual.
- Reglas complejas de combinacion de descuentos.
- Atribucion financiera exacta garantizada.
- Automatizaciones multicanal fuera de Shopify.
- UI compleja antes de cerrar modelos y contratos de datos.
